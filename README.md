# DataFusion WASM

Apache Datafusion that works on `wasm32-unknown-unknown`.

# Install

This repo is published to npm: https://www.npmjs.com/package/datafusion-wasm. Use the following command to add dependence:

```bash
npm i datafusion-wasm
```
